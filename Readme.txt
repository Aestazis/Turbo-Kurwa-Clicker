Turbo Kurwa Clicker 2.4
------------------------------------------------------------------------------------------------------------------------
Программа для имитации бурной деятельности. 
Позволяет записывать движения мышки и клики правой кнопкой, а затем их воспроизводить заданное количество раз.
------------------------------------------------------------------------------------------------------------------------
Инструкция: 
1. Запустите приложение clicker_2.4
2. Нажмите кнопку "Новая запись". Программа начнет запись ваших движений и кликов мышки. 
3. Для остановки записи нажмите "Esc". В окне "Записи" появится ваша запись.
4. Выберите нужно запись и нажмите кнопку "Воспроизвести". 
5. В окне "Циклы воспроизведения" укажите сколько раз должна повториться запись. Значение 0 запускает бесконечный цикл.
Курсор начнет движение и повторение ваших действий при записи.
6. Для остановки воспроизведения нажмите пробел. 
7. Для удаления записи выберите запись и нажмите "Удалить".

Горячие клавиши: 

ESC - остановка записи.
SPACE - остановка воспроизведения.
------------------------------------------------------------------------------------------------------------------------
P.S. 
Антивирус может ругаться на exe файл clicker_2.4. Это происходит из за особенностей компиляции через Pyinstaller.
